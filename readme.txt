Prosperity Scriptures. Verses related to prosperity from the bible.

== Description ==

This plugin will display random biblical scriptures related to prosperity on every admin page. Prosperity scriptures tell us we can be wealthy, have prosperity, and live in abundance. These are daily reminders help you create a prosperous mindset.

== Installation ==

1. Upload the `prosperity` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
